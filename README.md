# PG2: PRÁCTICA 3

## API DE SOUTH PARK

### Descripcion
Esta API proporciona información detallada sobre la serie **South Park**. Con esta API, los usuarios pueden acceder a datos estructurados sobre las siguientes entidades:


### 1. Personajes
- Nombre
- Edad
- Género
- Ocupación
- Descripción
- Primera aparición

### 2. Lugares 
- Nombre
- Tipo
- Descripcion

### 3. Temporadas
- Numero de temporada
- Cantidad de episodios
- Año de estreno

### 4. Episodios 
- Título
- Temporada
- Episodio
- Fecha de emisión
- Sinopsis
